# SO-100 Leader CAD

Printable STL parts for the SO-100 leader arm used by this repository's
teleoperation setup.

The STL format does not encode physical units. Preserve the source CAD/export
unit convention when slicing or importing these files, and verify scale against
the real leader hardware before printing.

| File | Upstream filename |
| --- | --- |
| `motor_holder.stl` | `Motor_Holder.stl` |
| `motor_holder_2.stl` | `Motor_Holder_2.stl` |
| `rotation_pitch.stl` | `Rotation_Pitch.stl` |
| `under_arm.stl` | `Under_Arm.stl` |
| `upper_arm.stl` | `Upper_Arm.stl` |
| `wrist_roll_pitch.stl` | `Wrist_Roll_Pitch.stl` |

The original names are recorded here so future CAD revisions can be matched
without keeping duplicate binaries at the repository root.
